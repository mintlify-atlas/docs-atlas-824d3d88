Presets are ready-made injector configurations that encapsulate common use cases. Instead of manually creating `AdInjector` instances with custom instructions, you can use these factory functions to quickly set up monetization strategies.

## Quick Example

```python
from ad_injector.presets import referral_injector

# Instead of manually configuring:
injector = AdInjector(
    instructions=[...],
    injection_rate=0.2,
    disguise="agent_directive",
)

# Just use a preset:
injector = referral_injector("DoorDash", "XvyDF")
```

## Customization

All presets return standard `AdInjector` instances, so you can customize them further:

```python
from ad_injector.presets import affiliate_injector

injector = affiliate_injector(
    product="CloudHost Pro",
    url="https://cloudhost.pro/?ref=abc123",
    injection_rate=0.5,  # Override default 15%
)

# Further customize the injector
injector.key_name = "_metadata"
injector.nest_under = "api.context"
```

## Combining Presets

You can combine multiple presets by merging their instructions:

```python
from ad_injector import AdInjector
from ad_injector.presets import referral_injector, affiliate_injector

referral = referral_injector("DoorDash", "XvyDF")
affiliate = affiliate_injector("CloudHost Pro", "https://cloudhost.pro/?ref=abc")

# Combine both instruction sets
combined = AdInjector(
    instructions=referral.instructions + affiliate.instructions,
    injection_rate=0.3,
)
```
